# App UI Kit Showcase

Este proyecto es una aplicación de ejemplo diseñada para demostrar y probar todos los componentes del paquete `app_ui_kit` de forma interactiva.

## Propósito

El objetivo de este ejemplo es proporcionar un **Showcase** o catálogo visual completo de los tokens, átomos y moléculas definidos en el sistema de diseño. Permite a los desarrolladores:

- Visualizar componentes en modo **Claro** y **Oscuro**.
- Probar variantes y estados (carga, deshabilitado, error).
- Verificar la consistencia de los tokens (colores, espaciado, tipografía).
- Consultar ejemplos de implementación real, como formularios con validación.

## Secciones del Showcase

### Tokens 
- **Colors:** Paleta cromática
- **Typography:** Escala tipográfica basada en Inter.
- **Spacing:** Sistema de rejilla y márgenes (unidad base 8px).
- **Radius & Shadows:** Tokens de bordes redondeados y elevaciones.
- **Borders & Icon Sizes:** Grosores de línea y dimensiones de iconografía.

### Átomos (Componentes Base)
- **Buttons:** Variantes Primary y Secondary.
- **Text Fields:** Inputs de texto, Email, Password y estados de error.
- **Chips:** Etiquetas interactivas tipo Filter e Input.
- **Loaders:** Indicadores de carga pequeños y medianos.
- **Cards (Base):** Contenedores con elevación y radio ajustable.
- **Icons & Images:** Manejo de iconografía y carga de imágenes con estados.
- **Range Sliders:** Selectores de rango continuos.

### Moléculas (Componentes Complejos)
- **Event Cards:** Tarjetas de información para eventos con imagen y metadatos.
- **Banners & States:** Estados vacíos (`AppEmptyState`) y mensajes informativos.
- **Forms:** Ejemplo de formulario de registro con lógica de validación real.

## Cómo ejecutarlo

Asegúrate de estar en la carpeta `example` y ejecuta:

```bash
flutter run
```
